<Update label="2024-09-20" description="v2.0.0" tags={["breaking", "feature"]}>

## New Features

- Introduced SF Matrix v2.0 with multi-asset correlation analysis for diversified portfolios.
- New API endpoints for programmatic strategy deployment.

<CodeGroup tabs="JavaScript,Python">
```javascript
const response = await fetch('https://api.example.com/v2/strategies', {
  method: 'POST',
  headers: { 'Authorization': 'Bearer YOUR_TOKEN' },
  body: JSON.stringify({ name: 'SF Matrix Pro', matrixVersion: '2.0' })
});
```

```python
import requests
response = requests.post(
    'https://api.example.com/v2/strategies',
    headers={'Authorization': 'Bearer YOUR_TOKEN'},
    json={'name': 'SF Matrix Pro', 'matrixVersion': '2.0'}
)
```
</CodeGroup>

## Breaking Changes

- Deprecated legacy SF Matrix v1 endpoints. Migrate using the guide below.

<Steps>
  <Step title="Update Config" icon="settings">
    
```json
{
  "sfMatrix": {

    "enabled": true
  }
}
```

  </Step>
  <Step title="Test Backtest" icon="play">

  </Step>
  <Step title="Deploy" icon="rocket">

  </Step>
</Steps>

</Update>
